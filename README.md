🛒 The Online Store
A clean, responsive static e-commerce storefront showcasing a curated collection of fashion and accessories. Built with a focus on simplicity and a modern user interface.
🔗 Live Site
View the deployed store here: https://jprhs11.github.io/STATIC-WEBSITE-DEPLOYMENT/
🛍️ Featured Products
The store currently features a variety of essential items:
Apparel: Denim Jackets, Classic T-Shirts, and cozy Scarves.
Accessories: Leather Bags, Sunglasses, and durable Belts.
Wearables: Classic Watches and stylish Sneakers.
🛠️ Technical Features
Grid Layout: A 5-column responsive product grid for optimal browsing.
Call to Action: Integrated "Buy" buttons for every product.
Static Deployment: Lightweight and fast-loading, optimized for GitHub Pages.
🚀 Local Setup
Clone the repository:
bash
git clone https://github.com/jprhs11/STATIC-WEBSITE-DEPLOYMENT.git

Launch:
Open index.html in any modern web browser to view the store locally.
📦 GitHub Pages Deployment
To host this store for free on GitHub Pages:
Push your code to the main branch.
Go to Settings > Pages in your GitHub repository.
Under Build and deployment, set the source to Deploy from a branch.
Select main and click Save.
